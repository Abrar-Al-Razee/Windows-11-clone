
# 🪟 Windows 11 Home Screen Clone

A sleek and responsive clone of the Windows 11 Home Screen built using **HTML**, **CSS**, and **JavaScript**. This project mimics the look and feel of the Windows 11 desktop, complete with taskbar, start menu, and draggable icons.

![screenshot](https://hc-cdn.hel1.your-objectstorage.com/s/v3/88963b0e3dade5bf32b8ef82740d1424c4da505b_screenshot_2025-07-20_at_12.20.58___am.png) 

---

## 🚀 Features

- 🖥️ Desktop layout inspired by Windows 11
- 🪟 Start Menu with modern UI
- 🔲 Taskbar with time, widgets, and shortcuts

---

## 🛠️ Built With

- **HTML5**
- **CSS3** (Flexbox, Grid, Animations)
- **JavaScript** (DOM manipulation, Events)

## 📁 Getting Started

### 🔧 Installation

1. Clone the repository:

```bash
git clone https://github.com/Abrar-Al-Razee/Windows-11-clone
```
Navigate into the project directory:
```
cd windows-11-home-screen
```
Open index.html in your browser: -> start index.html

💡 You can also use Live Server in VS Code for real-time reload.

✨ Customization
Edit index.html to add or remove apps

Modify style.css to change the theme or layout

Use script.js to add interactivity (drag-drop, open apps, etc.)
